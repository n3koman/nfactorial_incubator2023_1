1. Для начала, чтобы работать с этой программой надо установить MongoDB, чтобы создать локальную базу данных и для того, чтобы работать с NoSQL.
2. В приложении MongoDBCompass создать локальную БД, чтобы был доступ. 
3. После этого запустить питон файл.
4. Сразу при запуске для пользователя будет выбор: перейти уже в существующую базу данных, либо создать новый.
5. После этого создать коллекцию, чтобы сохранять туда поля
6. После пользователю откроются несколько выборов, можно добавлять, изменять, сохранить, удалить данные. Но чтобы для начала сохранить БД нужно добавить файлы
7. Также при сохранении результатов файл имеет расширение .csv, которое можно переконвертировать в таблицу Excel.
